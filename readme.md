# Informatics Results Aggregator

Расширение для браузера для агрегации результатов контестов с informatics.msk.ru

## Установка

1. Откройте `chrome://extensions/` (или `edge://extensions/`)
2. Включите "Режим разработчика"
3. Нажмите "Загрузить распакованное расширение"
4. Выберите папку `informatics_results_extension`

## Использование

1. Откройте страницу с результатами контеста на informatics.msk.ru
2. Расширение автоматически сохранит данные
3. Нажмите на иконку расширения для просмотра списка
4. Нажмите "Сводка" для полной таблицы

## Возможности

- Автоматический парсинг таблиц результатов
- Разделение на типы: "На туре" иk "Дорешка"
- Ссылки на каждую задачу
- Поиск по имени участника
- Экспорт в CSV
