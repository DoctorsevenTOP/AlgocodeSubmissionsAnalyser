# Informatics Results Aggregator

Расширение для браузера для агрегации результатов контестов с informatics.msk.ru

## Установка

1. Откройте `://extensions/`
2. Включите "Режим разработчика"
3. Нажмите "Загрузить распакованное расширение"
4. Выберите папку `informatics_results_extension`

## Использование

1. Войдите в свой аккаунт на informatics.msk.ru
2. Нажмите на иконку расширения, чтобы открыть список задач для парсинга.
3. Вбейте название и список ID задач, потом нажмите на кнопку "Сканировать".
4. Дождитесь завершения сканирования, откройте таблицу кнопкой "Открыть сводку"

Пример корректного списка ID:
```
Тур 1
112233
223344
=
Тур 2
112234
```

## Возможности

- Автоматический парсинг таблиц результатов
- Разделение на типы: "На туре" и "Дорешка"
- Ссылки на каждую задачу
- Экспорт в XLSX
